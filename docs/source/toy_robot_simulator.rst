Toy Robot Simulator Application
=============================================

This top-level script serves as the entry point to the Toy Robot Simulator application. To run the application,
simply run the Python script on a terminal using::

   Python3 /path/to/package/toy_robot_simulator/toy_robot_simulator.py

When run, an instance of the Requester object is created and its run() method is called. This will initiate the
interactions with the user through the console using Standard Input and Standard Output.
